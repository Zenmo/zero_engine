==========

This repository contains the 'engine' part of the Zenmo Zero models; the calculation model of the energy system simulation. The implementation is done in AnyLogic. 

In order to use the Zero Engine, you need to bootstrap the Agents. You can do this from another (new) AnyLogic project. 

Optionally works together with:

* [Zero Results UI](https://github.com/Zenmo/zero_results_UI)
* [Zero Configurator](https://github.com/Zenmo/zero)
